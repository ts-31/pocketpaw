## Overview

Returns the current health engine summary. The response includes the computed `overall_status`, the number of checks that have been run, any active issues, and the timestamp of the last check.

This endpoint does **not** trigger a new check run — it returns cached results from the most recent startup check or heartbeat. To force a fresh check, use [`POST /api/health/check`](/api/run-health-check).

<Callout type="info">
This endpoint requires no authentication. Health status is considered non-sensitive system metadata.
</Callout>

## Response

<ResponseField name="status" type="string">
  Overall system health: `healthy`, `degraded`, or `unhealthy`. Computed from individual check results — any critical failure makes the system unhealthy, any warning makes it degraded.
</ResponseField>

<ResponseField name="check_count" type="integer">
  Total number of health checks that have been run.
</ResponseField>

<ResponseField name="issues" type="array">
  List of check results that are **not** `ok`. Each issue is an object with:

  <ResponseField name="check_id" type="string">Unique identifier for the check (e.g. `api_key_primary`, `llm_reachable`)</ResponseField>
  <ResponseField name="name" type="string">Human-readable check name (e.g. "Primary API Key")</ResponseField>
  <ResponseField name="category" type="string">Check category: `config`, `connectivity`, or `storage`</ResponseField>
  <ResponseField name="status" type="string">Check status: `warning` or `critical`</ResponseField>
  <ResponseField name="message" type="string">Description of the issue</ResponseField>
  <ResponseField name="fix_hint" type="string">Suggested fix action</ResponseField>
  <ResponseField name="timestamp" type="string">ISO 8601 timestamp of when the check ran</ResponseField>
</ResponseField>

<ResponseField name="last_check" type="string">
  ISO 8601 timestamp of the most recent check run. Empty string if no checks have been run yet.
</ResponseField>

  </Tab>
  <Tab title="JavaScript">

    const data = await response.json();

    data = response.json()
    print(data["status"])

</RequestExample>
